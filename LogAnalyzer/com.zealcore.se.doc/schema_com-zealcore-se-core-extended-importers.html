Log Analyzer Importers

Identifier:
com.zealcore.se.core.extendedimporters

Since:
Log Analyzer 1.7

Description:
This is the new extension point for all Log Analyzer importers. A Log Analyzer importer is a parser that reads a log file and creates the corresponding events and artifacts.

Configuration Markup:

<!ELEMENT importer (EMPTY*)>

<!ATTLIST importer

id          CDATA #REQUIRED

name        CDATA #IMPLIED

description CDATA #IMPLIED

class       CDATA #REQUIRED>

Definition of an importer for importing log files into the Log Analyzer.


  • id - The unique identifier for the importer.
  • name - The human-readable name of the importer.
  • description - The human-readable description of the importer.
  • class - The fully qualified name of a public class with a no-arguments constructor that implements the com.zealcore.se.core.IExtendedImporter interface.

Examples:
See the examples in the "Writing an extended importer for Log Analyzer" chapter in this manual.

Supplied Implementation:
The SimpleExtendedImporter class in the "Writing an extended importer for Log Analyzer" chapter in this manual is provided as example importer.
