Log Analyzer Importers

This extension point is deprecated

Identifier:
com.zealcore.se.core.importers

Since:
Log Analyzer 1.4

Description:

WARNING: This extension point is DEPRECATED.
Do not use this extension point, it will be removed in future versions of this product. Instead, use the extension point com.zealcore.se.core.extendedimporters.

This is the extension point for all Log Analyzer importers. A Log Analyzer importer is a parser that reads a log file and creates the corresponding events and artifacts.

Configuration Markup:


The importer element is deprecated

<!ELEMENT importer (EMPTY*)>

<!ATTLIST importer

id          CDATA #REQUIRED

name        CDATA #IMPLIED

description CDATA #IMPLIED

class       CDATA #REQUIRED>

Definition of an importer for importing log files into the Log Analyzer.


  • id - The unique identifier for the importer.
  • name - The human-readable name of the importer.
  • description - The human-readable description of the importer.
  • class - The fully qualified name of a public class with a no-arguments constructor that implements the com.zealcore.se.core.IImporter interface.

Examples:
See the examples in the "Writing an importer for Log Analyzer" chapter in this manual.

Supplied Implementation:
The MinimalImporter and SimpleImporter classes in the "Writing an importer for Log Analyzer" chapter in this manual are provided as example importers.
