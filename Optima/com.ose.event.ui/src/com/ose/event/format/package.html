This package contains readers and writers for event action XML files, readers and writers for event XML files and event dump files, and converters between event XML files and event dump files and vice versa.

The event action XML file format is defined by the eventaction.dtd DTD in the com.ose.event.ui plugin. The event XML file format is defined by the event.dtd DTD in the com.ose.event.ui plugin. The event dump file format is an application of the Optima dump file format.
