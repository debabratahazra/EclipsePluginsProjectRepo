This package contains readers and writers for profiled process settings XML files, readers and writers for profiling report XML files and profiling report dump files, and converters between profiling report XML files and profiling report dump files and vice versa.

The profiled process settings XML file format is defined by the process.dtd DTD in the com.ose.prof.ui plugin. The profiling report XML file format is defined by the report.dtd DTD in the com.ose.prof.ui plugin. The profiling report dump file format is an application of the Optima dump file format.
